# 문항 구성과 성향 안내 근거

이 문서는 `나의 공부 스타일을 탐색하는 시간`이 어떤 자기조절학습 연구를 참고했고, 앱 내부 로직이 어떻게 작동하는지 짧게 설명하기 위한 공개 안내문이다.

## 먼저 분명히 할 점

- 이 도구는 심리검사, 표준화 진단, 상담·임상 평가가 아니다.
- 결과는 학생의 고정된 성격이나 능력을 말하지 않는다.
- 모든 성향 설명은 현재 답변을 바탕으로 다음 학습전략을 고르는 코칭 출발점이다.
- 학생 간 순위, 등급, 우열 비교를 목적으로 쓰지 않는다.

## 왜 5가지 축인가

문항은 자기조절학습 연구에서 반복적으로 다루는 전략을 중학생 수업 언어로 바꾼 것이다.

| 앱의 축 | 수업용 의미 | 연구 근거와의 연결 |
|---|---|---|
| 계획 세우기 | 목표를 정하고 공부 순서를 잡는 힘 | 학습 전 목표 설정과 계획 세우기 |
| 실행 유지하기 | 시작한 공부를 이어가는 힘 | 노력 조절, 집중 유지, 시간·환경 관리 |
| 이해 방법 찾기 | 개념을 나에게 맞는 방식으로 이해하는 힘 | 인지 전략, 자기설명, 정교화 |
| 점검하고 고치기 | 이해 여부를 확인하고 방법을 수정하는 힘 | 메타인지 점검, 피드백 활용, 성찰 |
| 질문과 도움 활용하기 | 막힐 때 자료·사람·AI를 적절히 활용하는 힘 | 자원 관리, 도움 요청, 학습 자원 활용 |

이 5축은 학생에게 연구 용어를 그대로 보여주기보다, 수업 중 바로 이해하고 행동으로 옮길 수 있는 말로 바꾼 구조다.

## 문항과 점수 로직

앱은 16문항으로 구성된다. 12문항은 4점 동의형이고, 4문항은 실제 공부 장면을 고르는 상황형 문항이다. 중립 선택지는 두지 않아 학생이 자신의 현재 행동에 더 가까운 쪽을 고르게 했다.

응답은 5축 점수로 합산된다. 이후 문항 구성에서 나올 수 있는 축별 최소·최대값을 기준으로 0~100 범위로 정규화한다. 학생 화면에서는 숫자보다 `강점`, `균형`, `성장 포인트` 라벨을 중심으로 보여준다.

## 코칭 프로필 로직

앱에서 성향으로 보여주는 8개 이름은 논문에서 그대로 가져온 학술 유형명이 아니다. 자기조절학습의 주요 요소를 수업용으로 재구성한 코칭용 대표 프로필이다.

앱은 학생의 5축 점수와 8개 기준 프로필의 거리를 비교해 가장 가까운 프로필을 찾는다. 단, `균형 조율형`은 평균·표준편차·최저 축 조건을, `기반 정리형`은 평균·최고 축 조건을 추가로 확인해 너무 쉽게 선택되지 않도록 한다. 1위와 2위가 매우 가까우면 보조 성향을 함께 설명에 반영한다.

전체 응답공간도 점검했다. `npm run logic:distribution`은 가능한 응답 조합 `10,485,760,000`개를 원점수 상태로 압축해 실제 분류 로직을 적용한다. 현재 기준에서 모든 성향은 3% 이상 나타나고, 단일 성향은 30%를 넘지 않는다.

| 성향 | 전체 응답 조합 비율 |
|---|---:|
| 전략 설계형 | 22.85% |
| 실행 추진형 | 19.00% |
| 개념 탐색형 | 17.57% |
| 자원 활용형 | 14.10% |
| 기반 정리형 | 9.62% |
| 점검 성장형 | 7.80% |
| 루틴 안정형 | 6.04% |
| 균형 조율형 | 3.03% |

이 검증은 특정 성향이 사실상 나오지 않거나 한 성향으로 과도하게 몰리는 문제를 줄이기 위한 개발 검증이다. 표준화 검사 규준을 의미하지는 않는다.

## 사용 권장 방식

결과는 학생에게 “너는 이런 사람”이라고 말하기 위한 자료가 아니다. 수업에서는 “지금 답변을 기준으로 어떤 전략부터 실험해볼까?”를 묻는 자료로 사용하는 것이 적절하다.
