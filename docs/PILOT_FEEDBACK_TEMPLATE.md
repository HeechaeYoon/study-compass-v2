# Educator Pilot Feedback Template

Use this template after a classroom or educator pilot of Study Compass V2. The goal is to improve usability, clarity, accessibility, and classroom fit **without collecting student data**.

## Privacy rule

Do **not** include any real student information in this document, issues, pull requests, screenshots, logs, or copied prompts.

Do not collect or paste:

- student names or nicknames
- student answers
- student memos
- generated prompt text copied from a real student session
- result screenshots that can identify a student
- school/class identifiers that can identify a specific group
- contact details or sensitive personal information

Use only aggregated observations, synthetic examples, or generalized notes.
